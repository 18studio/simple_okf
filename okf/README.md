# OKF bundles

Эта директория содержит OKF knowledge bundles.

Основной bundle шаблона:

```text
okf/platform-system/
```

Правила:

- concept-файлы — обычные `.md` с YAML frontmatter;
- `index.md` — навигация, не concept;
- `log.md` — журнал, не concept;
- связи — Markdown links между concept-файлами;
- предпочтительны относительные ссылки.
